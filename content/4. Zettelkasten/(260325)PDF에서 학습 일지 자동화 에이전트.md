# PDF → 학습 일지 자동화 에이전트

> 오늘 직접 해본 것을 에이전트 워크플로로 만들면?

![[idea-pdf-automation.svg]]

## 아이디어

오늘 Claude Code로 수동으로 한 작업 — PDF 읽기 → 노트 작성 → 핵심 포인트 정리 → 학습 일지 업데이트 — 를 에이전트 워크플로로 완전 자동화한다.

PDF 파일 경로 하나만 던져주면 30분 후 Obsidian에 노트와 학습 일지가 완성되는 것이 목표.

## 워크플로 설계

```
[입력] PDF 파일 경로
    ↓
[에이전트 1] PDF 파서
    - 텍스트 추출
    - 저자/날짜/출처 메타데이터 파싱

[에이전트 2] 노트 생성기
    - 요약 1줄
    - 핵심 포인트 3개 (계층형)
    - 인사이트 작성
    - article 템플릿에 맞게 frontmatter 구성

[에이전트 3] 학습 일지 업데이터
    - 오늘 날짜의 학습 일지 확인/생성
    - 읽은 것 섹션에 링크 추가
    - 오늘의 인사이트 업데이트

[출력] Obsidian 노트 + 학습 일지 자동 생성
```
